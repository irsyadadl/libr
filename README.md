### Bootstrap Breeze
Follow these steps, and you good to go!

```bash
git clone https://github.com/teamparsinta/breeze-bootstrap.git 'project'
cd project
composer install
cp .env.example .env
php artisan key:generate
npm install
npm run dev
```
